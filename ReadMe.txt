Steps To Run Console Based Library
Main Source code file- Library.java
Main Class file- Library.java
1.Enter the port location and password in the details.txt file
2.To set the password for library, use User-Pass.txt
	Note: For protection, your Password get reverse every time you run the program.
3.Open Cmd, set the path to the location of Library.jar file 
4.Now run the Lib.bat file using Cmd.

Note: You have to set the class path of the Mysql-Java Connector.
